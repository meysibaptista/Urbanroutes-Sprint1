# Projeto 1: Teste de Regressão do Urban Routes

## 📌 Descrição do Projeto
Neste projeto, eu conduzi um teste de regressão de um aplicativo de rotas chamado **Urban Routes**, com o qual você continuará trabalhando nos próximos sprints.

O **Urban Routes** é um aplicativo que cria rotas e calcula o tempo e o custo da viagem para diferentes tipos de transporte.

---

## 🧭 Funcionalidades do Aplicativo

A interface do aplicativo é simples e contém:

- Dois campos de endereço:
  - **De**
  - **Para**
- Três modos de viagem:
  - Optimal
  - Flash
  - Personal
- Tipos de transporte disponíveis:
  - Carro próprio
  - A pé
  - Táxi
  - Bicicleta
  - Scooter
  - Compartilhamento de carro

---

## ⚙️ Funcionamento

1. O usuário insere os endereços nos campos **"De"** e **"Para"**.
2. O sistema converte esses endereços em pontos **A** e **B**.
3. O aplicativo calcula:
   - Tempo total da viagem
   - Custo total da viagem  
4. O cálculo é feito com base em um algoritmo específico.

---

## 🎯 Objetivo do Projeto

A equipe de QA preparou casos de teste para o teste de regressão do Urban Routes.

Seu objetivo é:

- Executar os casos de teste
- Definir um status apropriado para cada teste

---

## ⚠️ Importância do Teste

Como qualquer outro aplicativo, o Urban Routes pode apresentar bugs.

Se os usuários tiverem uma experiência ruim:
- Podem deixar avaliações negativas
- Podem migrar para aplicativos concorrentes

Por isso, é essencial identificar e corrigir problemas rapidamente.
